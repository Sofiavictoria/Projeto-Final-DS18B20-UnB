# Projeto de Calibração do Sensor DS18B20

Bem-vinda(o)! Esta é a documentação do projeto final da disciplina **Instrumentação Eletrônica para Engenharia — UnB (2025/2)**.

**Objetivo:** calibrar o sensor DS18B20, compará-lo com um sensor de referência e implementar uma compensação digital baseada nos desvios observados.

- [Baixar relatório (PDF)](assets/Relatorio_Projeto_Final_DS18B20.pdf)


## Sobre o Projeto

Este trabalho apresenta a calibração estática do sensor digital **DS18B20**, realizada com o objetivo de avaliar seu desempenho real, comparar com um padrão de referência e aplicar uma correção digital simples.

A aplicação motivadora é o **monitoramento térmico de aquários**, que exige estabilidade e sensores confiáveis em ambientes úmidos.

O relatório completo inclui:

- descrição do experimento;  
- metodologia de calibração;  
- análise dos dados;  
- modelo de compensação;  
- conclusões.
