# Sobre o Projeto

O sensor **DS18B20** foi escolhido por ser digital, simples de integrar e amplamente usado. 
O experimento consistiu em:

- montagem do circuito com Arduino Uno;
- coleta de leituras em 3 cenários (água gelada, ambiente, água quente);
- comparação com termômetro clínico como referência;
- ajuste por regressão e aplicação de compensação.
